# Bonetflix

Bonetflix helps you discover the best movies and series on Netflix using IMDb and Filmaffinity ratings.

🌐 Website:
https://bonetflix.com

Bonetflix includes:

- A website with rankings of Netflix movies and series
- A browser extension that displays ratings directly on Netflix

---

## Website

The Bonetflix website lists the best movies and series available on Netflix ranked by ratings.

Features:

- IMDb and Filmaffinity rankings
- Top movies and series
- Genre filtering
- Country-specific catalogs
- Updated Netflix availability

Visit:

https://bonetflix.com

---

## Browser Extension

The Bonetflix extension shows ratings directly inside Netflix.

Features:

- IMDb ratings on Netflix
- Filmaffinity ratings on Netflix
- Works automatically
- Lightweight and fast

Install:

Chrome:
https://chromewebstore.google.com/detail/bonetflix/gigcpmnhlgefalllpfihefpnpmbpblal

Firefox:
https://addons.mozilla.org/firefox/addon/bonetflix/

Edge:
https://microsoftedge.microsoft.com/addons/detail/bonetflix/gmhadgjeicbchegcdcplpfimdceoafdg

---

## How It Works

Bonetflix combines rating data from:

- IMDb
- Filmaffinity

and matches them with Netflix titles to help users choose what to watch faster.
